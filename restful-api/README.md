# restful-api

## task_02_requests
a module that fetches posts from a public API using the requests library, prints their titles and saves them to a CSV file

## task_03_http_server
a simple HTTP API server built with the http.server module that handles GET requests on multiple endpoints returning plain text or JSON responses

## task_04_flask
a Flask API with endpoints to manage users, supporting listing all usernames, retrieving a specific user, and adding new users via POST requests

## task_05_basic_security
a Flask API secured with both HTTP Basic Authentication and JWT token-based authentication, including an admin-only endpoint with role-based access control
