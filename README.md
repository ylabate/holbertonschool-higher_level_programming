# holbertonschool-higher_level_programming

Small collection of introductory Python exercises for the Holberton curriculum. Each folder below includes a short README and runnable scripts to practice core language concepts.

## Structure
- [python-hello_world](python-hello_world/): first steps with printing, strings, and basic formatting.
- [python-if_else_loops_functions](python-if_else_loops_functions/): conditionals, loops, simple functions, and classic fizzbuzz.
- [python-import_modules](python-import_modules/): importing functions, variables, and exploring module attributes.
- [python-data_structures](python-data_structures/): exercises on lists, tuples, and strings.
- [python-exceptions](python-exceptions/): handling exceptions and errors in Python.
- [python-classes](python-classes/): introduction to classes and objects.
- [python-more_data_structures](python-more_data_structures/): working with dictionaries and sets.
- [python-test_driven_development](python-test_driven_development/): introduction to test-driven development with doctest.
- [python-more_classes](python-more_classes/): deeper dive into classes and objects.
- [python-everything_is_object](python-everything_is_object/): exploring object references, mutability, and memory in Python.
- [python-inheritance](python-inheritance/): introduction to inheritance in Python.
- [python-input_output](python-input_output/): reading from and writing to files, working with JSON.
- [python-abc](python-abc/): abstract base classes and interfaces in Python.
- [python-serialization](python-serialization/): serializing and deserializing data structures.
- [python-server_side_rendering](python-server_side_rendering/): introducing Flask and Jinja templates.
- [python-object_relational_mapping](python-object_relational_mapping/): mapping Python objects to SQL databases using SQLAlchemy.
- [javascript-warm_up](javascript-warm_up/): introduction to JavaScript basics.
- [javascript-dom_manipulation](javascript-dom_manipulation/): manipulating the DOM using JavaScript.
- [SQL_introduction](SQL_introduction/): introduction to SQL and relational databases.
- [SQL_more_queries](SQL_more_queries/): more advanced SQL queries and joins.
- [restful-api](restful-api/): building a RESTful API.

## How to run
1. Use Python 3: `python3 script.py`
2. Scripts are self-contained; no extra dependencies are required.
3. Check each subfolder README for script-specific notes.
