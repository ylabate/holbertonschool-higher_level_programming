Test for FlyingFish Class
===========================

The FlyingFish class inherits from both Fish and Bird classes and overrides their methods.

>>> from task_04_flyingfish import Fish, Bird, FlyingFish

Test Fish class methods:

>>> fish = Fish()
>>> fish.swim()
The fish is swimming
>>> fish.habitat()
The fish lives in water

Test Bird class methods:

>>> bird = Bird()
>>> bird.fly()
The bird is flying
>>> bird.habitat()
The bird lives in the sky

Test FlyingFish class methods:

>>> flying_fish = FlyingFish()
>>> flying_fish.swim()
The flying fish is swimming!
>>> flying_fish.fly()
The flying fish is soaring!
>>> flying_fish.habitat()
The flying fish lives both in water and the sky!
